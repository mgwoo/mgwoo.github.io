Projects

Circuit Projects

RePlAce - RePlAce is a global placement tool for spreading and placing standard cells by minimizing wire-length. RePlAce solves the electrostatic PDEs for density calculation, which is a core algorithm for spreading the standard cells. I've rewritten the whole RePlAce as clean C++11 code based on OpenDB, which means RePlAce now fully supports LEF/DEF commercial formats in fast. (Verified with 7/12/14/28/45/65/130 nm technologies.). Routability-driven and Timing-driven implementations are ongoing with the clean structure. RePlAce has a major role in the 'global_placement' command in OpenROAD flow.
Timing-Driven RePlAce with OpenSTA API - Ported OpenSTA commercial timer engine to old RePlAce. Net-reweighting has been applied for timing critical nets. RePlAce can generate smaller WNS/TNS results compared to a commercial tool.
OpenDP; Fence Region-Aware Multi-Deck Standard Cell Legalization - Proposed fence-region and multi-deck-aware open-source detail placer [2]. Ported Si2 LEF/DEF parsers into OpenDP. Broadly used in Synthesis, Gate-sizing, and Clock Tree Synthesis (CTS). OpenDP has a major role in the 'legalize_placement' command in OpenROAD flow.
Triton Macro Place - ParquetFP based macro cell placer for OpenROAD. Generate a placed DEF with macro placements honoring halos, channels and cell row 'snapping'. Approximately ceil((#macros/3)^(3/2)) 'sets' corresponding to quadrisections of the initial placed mixed-size DEF are explored and packed using ParquetFP-based annealing. The best resulting floorplan according to a heuristic evaluation function is returned. TritonMP has a major role in the 'macro_placement' command in OpenROAD flow.
FLEX/BISON Verilog Parser for C++ - Modify the previous Ben-marshall's verilog parser to be successfully compiled under c++11/c++14 environments.
Verilog To DEF translator - Convert hierarchical/flat gate-level Verilog to a flat DEF file. Verilog-to-DEF is Helpful for both of floorplanner and placer. The converter is based on the OpenSTA's ConcreteNetwork API.
GRASP based Metaheuristics for Layout Pattern Classification - Proposed new methodology for Layout Pattern Calssification problem [1]
